<objective>
Fix the duplicated IntersectionObserver bug and establish a single, reliable animation system with accessibility guards.

Purpose: The current animation system has two bugs: (1) IntersectionObserver scripts are duplicated in FeatureCTA.astro and LandingCTA.astro instead of living in BaseLayout, and (2) the CSS hides content unconditionally (broken without JS) and applies motion in the reduced-motion block. This plan consolidates the observer into BaseLayout, adds .js-enabled CSS gating, and implements strict zero-motion for reduced-motion users.

Output: Four modified files -- global.css (refactored animation section), BaseLayout.astro (AnimationController script added), FeatureCTA.astro and LandingCTA.astro (duplicate scripts removed).
</objective>

- `is:inline` directive (not bundled) -- same pattern as Header.astro scroll shadow script
