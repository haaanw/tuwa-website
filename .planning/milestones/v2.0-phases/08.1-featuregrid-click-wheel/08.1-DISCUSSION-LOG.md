---

## Vision

| Option | Description | Selected |
|--------|-------------|----------|
| Scrollable wheel | Cards rotate in a carousel/wheel pattern | |
| Click-to-expand | Clicking a card expands it inline | |
| Interactive selector | One card active at a time, clicking cycles | |
| Other (free text) | "use the way that reminds people how they use ipod" | ✓ |

**User's choice:** iPod click wheel — the circular touch/click interface from the original iPod
**Notes:** User wants the FeatureGrid to evoke the iPod experience

---

## Visual Style

| Option | Description | Selected |
|--------|-------------|----------|
| Literal circle wheel | Visible circular ring with feature names/icons around circumference | ✓ |
| 3D carousel ring | Features as cards in a 3D perspective ring (Cover Flow style) | |
| Radial menu | Flat circle with feature icons as segments (pie menu) | |

**User's choice:** Literal circle wheel — closest to real iPod click wheel
**Notes:** User confirmed this is most iPod-like after asking

---

## Selection Behavior

| Option | Description | Selected |
|--------|-------------|----------|
| Center preview + link | Selected feature shows in center, click center to navigate | ✓ |
| Direct navigation | Selecting immediately navigates to deep-dive page | |
| Expand inline | Selected feature expands into card preview beside wheel | |

**User's choice:** Center preview + link

---

## Input Method

| Option | Description | Selected |
|--------|-------------|----------|
| Drag to rotate | Click/touch and drag along circle | |
| Click segments | Click feature name to select | |
| Both | Drag to rotate AND click to select | ✓ |

**User's choice:** Both — full iPod experience

---

## Mobile Behavior

| Option | Description | Selected |
|--------|-------------|----------|
| Same wheel, smaller | Scale wheel down, touch drag works naturally | ✓ |
| Fallback to bento | Keep bento grid on mobile, wheel on desktop only | |
| Vertical scroll wheel | Transform to vertical scrollable list on mobile | |

**User's choice:** Same wheel, smaller — consistent experience across breakpoints
