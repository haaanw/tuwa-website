```html
<script>
  import Lenis from 'lenis';
  import 'lenis/dist/lenis.css';

  // D-03: Disable on touch devices — native momentum is superior
  const isTouch = window.matchMedia('(pointer: coarse)').matches;
  const reducedMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches;

  if (!isTouch && !reducedMotion) {
    const lenis = new Lenis({
      autoRaf: true,
      lerp: 0.1,
      anchors: {
        offset: 64, // Sticky header height (src/components/Header.astro)
      },
    });

    // Expose for potential programmatic scrollTo access
    (window as any).__lenis = lenis;
  }
</script>
```

- `anchors.offset: 64` — accounts for sticky header so anchor targets aren't hidden
- Touch detection via `(pointer: coarse)` — reliable, no UA sniffing
- Module script (not is:inline) because Lenis requires ESM import
- No `scroll-behavior: smooth` in CSS (confirmed absent) — Lenis handles anchor scrolling

Do NOT add `scroll-behavior: smooth` to html/body in global.css (Pitfall 4 from RESEARCH.md).

Add a SECOND `<script>` tag (module) in BaseLayout.astro AFTER the Lenis script (before `</body>`). Per D-04, D-05, D-06:

```html
<script>
  // Magnetic CTA — cursor proximity pull effect (D-04, D-05, D-06)
  const RADIUS = 150;   // Activation radius in px
  const STRENGTH = 14;  // Max shift in px (10-16px range per D-04)

  function initMagnetic() {
    // D-06: No magnetic on touch devices
    if (window.matchMedia('(pointer: coarse)').matches) return;
    if (window.matchMedia('(prefers-reduced-motion: reduce)').matches) return;

    const buttons = document.querySelectorAll<HTMLElement>('.btn-cta');

    buttons.forEach((btn) => {
      btn.addEventListener('mousemove', (e: MouseEvent) => {
        const rect = btn.getBoundingClientRect();
        const cx = rect.left + rect.width / 2;
        const cy = rect.top + rect.height / 2;
        const dx = e.clientX - cx;
        const dy = e.clientY - cy;
        const dist = Math.sqrt(dx * dx + dy * dy);

        if (dist < RADIUS) {
          const pull = (1 - dist / RADIUS) * STRENGTH;
          const angle = Math.atan2(dy, dx);
          const tx = Math.cos(angle) * pull;
          const ty = Math.sin(angle) * pull;
          btn.style.transform = `translate(${tx}px, ${ty}px) scale(1.02)`;
        } else {
          btn.style.transform = '';
        }
      });

      btn.addEventListener('mouseleave', () => {
        // Clear inline transform — CSS transition (300ms var(--ease-interactive))
        // handles the smooth return to center
        btn.style.transform = '';
      });
    });

    // Pitfall 2 fix: reset all buttons if mouse exits viewport abruptly
    document.addEventListener('mouseleave', () => {
      buttons.forEach((btn) => {
        btn.style.transform = '';
      });
    });
  }

  initMagnetic();
</script>
```

Key implementation notes:
- `scale(1.02)` included in JS transform to match existing CSS hover state (Pitfall 5)
- On mouseleave, clearing `style.transform` lets CSS `transition: transform 300ms var(--ease-interactive)` (from Plan 01) animate smooth return
- Document mouseleave fallback handles fast mouse exit from viewport (Pitfall 2)
- Per-button mousemove listener (not document-level) — efficient for sparse buttons
- `else` branch in mousemove clears transform when cursor is in listener but outside radius
- D-05: Uses `.btn-cta` selector which captures ALL CTA buttons site-wide
