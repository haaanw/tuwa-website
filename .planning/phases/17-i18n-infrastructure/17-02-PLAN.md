<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| URL path -> locale | URL path determines locale at build time (static site); no runtime locale injection |
| Translation files -> HTML | Translation strings are embedded at build time; no user input in translation lookup |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-17-03 | Spoofing | getRelativeLocaleUrl | accept | Function validates locale against i18n config at build time; throws for unknown locales (Pitfall 6). Static site — no runtime locale spoofing possible. |
| T-17-04 | Information Disclosure | CJK font conditional loading | accept | Font files are public assets; conditional loading is a performance optimization, not a security boundary |
| T-17-05 | Denial of Service | Font file size | accept | @fontsource/noto-sans-sc uses unicode-range subsetting; browser downloads only needed character slices. No full CJK font dump. |
</threat_model>
