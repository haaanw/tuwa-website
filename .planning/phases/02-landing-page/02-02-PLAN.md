**Add the IntersectionObserver inline script** at the bottom of LandingCTA.astro (runs once, targets all `[data-animate]` elements on the page):
```html
<script is:inline>
  (function () {
    var observer = new IntersectionObserver(
      function (entries) {
        entries.forEach(function (entry) {
          if (entry.isIntersecting) {
            entry.target.classList.add('is-visible');
            observer.unobserve(entry.target);
          }
        });
      },
      { threshold: 0.15 }
    );
    document.querySelectorAll('[data-animate]').forEach(function (el) {
      observer.observe(el);
    });
  })();
</script>
```
This follows the established `is:inline` IIFE pattern from Header.astro scroll shadow script.
