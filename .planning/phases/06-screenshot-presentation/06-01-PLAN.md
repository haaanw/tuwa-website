**Props interface:**
```typescript
interface Props {
  src?: ImageMetadata;
  alt: string;
  placeholderLabel?: string;
  loading?: 'lazy' | 'eager';
  fetchpriority?: 'auto' | 'high' | 'low';
  widths?: number[];
  class?: string;
}
```

**Defaults:** `placeholderLabel = 'Coming soon'`, `loading = 'lazy'`, `fetchpriority = 'auto'`, `widths = [320, 640]`, `class` is optional.

**Component structure (outermost to innermost):**

1. **Outer wrapper div** — responsive fluid width:
   - Tailwind classes: `mx-auto w-full max-w-[260px] sm:max-w-[300px] lg:max-w-[320px]`
   - Also applies `className` prop if provided via template literal: `class={\`mx-auto w-full max-w-[260px] sm:max-w-[300px] lg:max-w-[320px] ${className ?? ''}\`}`

2. **Frame body div** — the iPhone body (class `device-frame`):
   - `position: relative`
   - `background-color: #1A1A1A` (D-02: near-black iPhone Space Black — hardcoded, not a design token)
   - `border-radius: 44px` (UI-SPEC outer radius)
   - `padding: 12px 10px 10px` (UI-SPEC frame border width — top is 12px for dynamic island area, sides/bottom 10px)
   - `box-shadow: inset 0 1px 0 rgba(255,255,255,0.08), 0 24px 64px rgba(28,25,21,0.22), 0 4px 16px rgba(28,25,21,0.12)` (metal edge reflection + premium depth shadow)

3. **Dynamic Island div** (inside frame body, before screen):
   - `aria-hidden="true"`
   - `position: absolute; top: 22px; left: 50%; transform: translateX(-50%)`
   - `width: 120px; height: 34px`
   - `background: #0D0D0D` (slightly darker than frame)
   - `border-radius: 20px; z-index: 2`

4. **Screen area div** (inside frame body, after dynamic island):
   - `border-radius: 36px` (UI-SPEC inner radius — 8px tighter than outer)
   - `overflow: hidden` (CRITICAL — prevents screenshot bleed past rounded corners, Pitfall 2)
   - `aspect-ratio: 9 / 19.5` (iPhone portrait ratio — matches existing ScreenshotBlock)
   - `position: relative; background: #000`

5. **Inside screen area — conditional render:**

   If `src` is provided (screenshot exists):
   ```astro
   <Image
     src={src}
     alt={alt}
     widths={widths}
     sizes="(min-width: 1024px) 320px, (min-width: 640px) 300px, 260px"
     loading={loading}
     fetchpriority={fetchpriority}
     style="width: 100%; height: 100%; object-fit: cover; display: block;"
   />
   ```
   NOTE: Do NOT include `format="webp"` or `quality={90}` — Astro 6 auto-generates WebP. Do NOT include `width={320}` — use `widths` array instead (D-08).

   If `src` is NOT provided (placeholder):
   ```astro
   <div
     role="img"
     aria-label={alt}
     style="
       width: 100%;
       height: 100%;
       display: flex;
       align-items: center;
       justify-content: center;
       background: linear-gradient(var(--color-surface-el), var(--color-divider));
       color: var(--color-text-3);
       font-size: var(--text-label);
       text-transform: uppercase;
       letter-spacing: var(--tracking-label);
     "
   >
     {placeholderLabel}
   </div>
   ```

6. **Home indicator div** (inside frame body, after screen area):
   - `aria-hidden="true"`
   - `width: 120px; height: 5px`
   - `background: rgba(255,255,255,0.3); border-radius: 3px`
   - `margin: 8px auto 0`
