## Decisions Made

- Used SVG `transform` attribute (`rotate(N, 240, 240)`) for drag rotation — avoids `transform-box` cross-browser bugs. CSS class `wheel-snap-transition` applied only for the final snap animation step.
- `iconSvg` values are static hardcoded string constants in the script — `temp.innerHTML` on a detached `createElementNS` SVG element is safe for parsing static path data (never user input, never inserted to document).
- Segment buttons positioned using percentage-based `left`/`top` values (`x/480*100%`) so they remain aligned with arc midpoints at all three responsive breakpoints without needing ResizeObserver.
- Pre-existing TSC error in `astro.config.mjs` (Vite plugin type mismatch — `Plugin<any>[]` not assignable to `PluginOption`) confirmed pre-existing before this plan via `git stash` test. Not caused by Phase 08.1 changes.

## Deviations from Plan
