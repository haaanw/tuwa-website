Implement hreflang tags (I18N-11), og:locale metadata (I18N-12), sitemap i18n annotations (I18N-13), and locale-specific 404 pages (I18N-14).

Purpose: Search engines need hreflang signals to serve the correct locale variant in search results. Social platforms use og:locale to render localized preview cards. The locale 404 pages prevent language-jarring fallbacks when users navigate to missing localized paths.
